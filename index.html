Code a Pookalam

Organized by Tinkerhub NSSCE

Create a digital pookalam using code and win exciting prizes!

Guidelines

  • Use any programming language of your choice.
  • Your code should generate a pookalam pattern.
  • Creativity and uniqueness will be given priority.
  • Submit your code along with a screenshot of the output.
  • Ensure originality – plagiarism will lead to disqualification.

Steps to Participate

  1. Register for the competition.
  2. Choose your favorite programming language.
  3. Write code to generate a pookalam design.
  4. Take a screenshot of your pookalam.
  5. Submit your code and screenshot before the deadline.

Examples

Python (Turtle)

Create colorful patterns using the Turtle library.

JavaScript (Canvas)

Draw geometric rangoli-like shapes with Canvas API.

C / C++

Generate ASCII art style pookalam designs.
